## &nbsp;&nbsp;Kyberon &nbsp;·&nbsp; Sovereign P2P Infrastructure

A Rust core engine (**~20 crates**) exploring how much of the centralized cloud can be replaced by a **verifiable, peer-to-peer mesh.** Deliberately **early** — the core is an alpha, single-node in-memory prototype — and every metric is re-measured, never hardcoded.

<div align="center">
<br/>

| &nbsp; | Module | Status | What's real today |
| :---: | :--- | :---: | :--- |
| 🔐 | **Ghost-Payments** | `kani-proven` | No underflow + mass conservation, verified by model checking |
| 📡 | **BLE Mesh** | `alpha` | X25519 + Kyber-768 hybrid handshake · 10/10 tests |
| 📻 | **LoRa** | `alpha` | Reed-Solomon FEC · 12/12 tests · HW not yet tested |
| 🔁 | **Proof-of-Relay** | `alpha` | Rewards + anti-replay · 8/8 tests |
| 🌐 | **sdk-relay-web** | `alpha` | WASM · ~225 KB gzipped · 11/11 tests |

<br/>
</div>

> **Honest by design.** Non-reproducible metrics were removed. Benchmarks are Criterion micro-benchmarks (single-thread, in-memory) — real distributed throughput will differ and is being re-measured.
